# 🤖 AI Virtual Friend

An AI-powered virtual friend that interacts with users in real-time using **face detection, emotion recognition, and intelligent conversation**. The system analyzes user emotions and responds accordingly to create a human-like and personalized experience.

---

## 🚀 Features

* 🎯 Face Detection & Recognition
* 😊 Emotion Detection using Deep Learning
* 💬 Smart Conversation System
* 🧠 Context-aware responses
* 🌐 Modular backend architecture
* 🗄️ Database integration for storing interactions

---

## 🛠️ Tech Stack

* **Language:** Python
* **AI/ML:** OpenCV, Dlib, DeepFace
* **Backend Architecture:** Modular Python Services
* **Database:** MongoDB
* **Libraries:** NumPy, Pandas

---

## 📁 Project Structure

```
AI-Virtual-Friend/
│
├── .env/              # Environment variables (not pushed)
├── config/            # Configuration files
├── core/              # Core logic / main controllers
├── database/          # Database connection & queries
├── models/            # ML / DL models
├── services/          # Business logic & services
├── utils/             # Helper functions
├── main.py            # Entry point of application
└── __pycache__/       # Cache files (ignored)
```

---

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/your-username/AI-Virtual-Friend.git
cd AI-Virtual-Friend
```

2. Create virtual environment:

```
python -m venv venv
venv\Scripts\activate   # Windows
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the project:

```
python main.py
```
